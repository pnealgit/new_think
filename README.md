#new_think

THIS REPOSITORY IS NO LONGER SUPPORTED. GO TO MNB_THINK for the latest.

A total javascript Markov Network Brain system.

Circles search for food using a network of logic gates.

My own take on networked brains ala Chris Adami's ideas. See: http://adamilab.msu.edu/markov-network-brains/

Also see http://devosoft.org/a-quick-introduction-to-markov-network-brains/

To run: 

  FIRE UP A SERVER:
    cd to the www directory
    python -m SimpleHTTPServer

  OPEN UP A WEB PAGE:
    localhost:8000
